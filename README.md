# Overview
This project began as a hackathon demo and has since evolved with additional features. It now serves as a portfolio piece designed to showcase skills in Angular and Node.js/Express.js. 

# Tech Stack
- Frontend: Angular
- Backend: Express.js, MongoDB

Backend repository: https://github.com/Nilo18/SkillMentor-Back/
  
# Key Features:
- Mentor registration and profile browsing
- Search and filtering for mentors by expertise
- Booking simulation
- Responsive UI for multiple devices
