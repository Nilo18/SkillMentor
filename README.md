# Overview
This project began as a hackathon demo and has since evolved with additional features. It now serves as a portfolio piece designed to showcase skills in Angular and Node.js/Express.js. Please note, it is not intended to be a fully functional website but rather a simulation of a Georgian online mentorship platform. As such, some inconsistencies may be present.

# Tech Stack
- Frontend: Angular
- Backend: Express.js, MongoDB
  
**Please note that the backend is hosted on a free tier web service so initial load time may take a while**
  
# Key Features:
- Mentor registration and profile browsing
- Search and filtering for mentors by expertise
- Booking simulation
- Responsive UI for multiple devices
